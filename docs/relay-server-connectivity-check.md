# 腾讯云 Relay 服务器连通性检查

目标服务器：`43.156.16.10`

检查时间：2026-07-04

## 已确认

- ICMP ping 可达，测试延迟约 `36ms`。
- `22/tcp` 可连接，说明 SSH 端口公网可达。
- `80/tcp` 可连接，`http://43.156.16.10` 返回 `HTTP/1.1 200 OK`，服务为 nginx。
- `443/tcp` 可连接，但 `https://43.156.16.10` TLS 握手失败，当前不能视为可用 HTTPS。

## 结论

服务器公网在线，适合作为 MVP relay 测试机。MVP 阶段可优先使用：

```text
ws://43.156.16.10/ws
```

或在安全组和系统防火墙放行后使用：

```text
ws://43.156.16.10:8080/ws
```

HTTPS/WSS 暂不作为 MVP 内测前置条件。正式外测、账号、支付、正式 token 或隐私数据上线前，需要配置域名和 TLS。

## 待确认

- SSH 登录账号、密钥或密码。
- 服务器系统版本、CPU 架构、Node.js 版本和可部署目录。
- 腾讯云安全组是否允许独立测试端口。
- nginx 配置目录和是否允许增加 `/ws` 反代。
